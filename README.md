⚔️ Bridge Duels AutoFarm — By Sxzly
Un script de autofarm moderno y optimizado para Bridge Duels, diseñado para maximizar eficiencia con una interfaz limpia y funcional.

✨ Features
🏆 Automatic Win Farming: Gana partidas de forma automática y continua.

📊 Real-Time Stats: Seguimiento de victorias y tiempo de ejecución.

⚡ Optimized Execution: Código ligero para evitar crasheos en sesiones largas.

🎨 Sleek UI: Interfaz intuitiva y fácil de usar.

💾 Data Persistence: Guarda tu progreso automáticamente.

📍 Auto-Detection: Detecta automáticamente si estás en el Lobby o en partida.

📦 Installation
Quick Load (Recommended)
Copia y pega este código en tu executor favorito:

Lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Sxzly/-BRIDGE-DUELS-AUTOFARM/main/script.lua"))()
Manual Install
Descarga el archivo script.lua desde el repositorio de GitHub.

Pégalo directamente en la carpeta de scripts de tu executor.

Ejecuta el script una vez dentro de Bridge Duels.

🎮 How to Use
Entra al modo Bridge Duels en BedWars.

Ejecuta el script mediante el loadstring proporcionado arriba.

El script gestionará el ciclo de juego:

⏳ Espera de seguridad inicial.

⚔️ Teletransportación o toque automático de la zona de victoria.

🔄 Re-queue automático para la siguiente partida.

🎨 UI & UX
Diseño Limpio: Interfaz minimalista que no obstruye la visión del juego.

Modo Privacidad: Opción para ocultar datos sensibles.

Acceso Rápido: Botones optimizados para activar/desactivar funciones al instante.

🐛 Troubleshooting
¿El script no carga?

Asegúrate de que tu executor sea compatible con loadstring y peticiones HTTP.

Verifica que estás dentro del juego correcto (Bridge Duels).

¿No gana las partidas?

Algunos executors gratuitos no soportan firetouchinterest. Intenta usar un executor de mayor nivel.
